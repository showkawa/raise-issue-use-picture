# 03 — 真机协议冒烟（格式定案 + 预算标定）

**What to build:** 一个可重复运行的冒烟脚本：对真实 Copilot 会话注入协议、发握手块、校验往返保真，并实测消息长度上限/会话预算/轮次上限，把结果回填为配置默认值；协议格式据此定案（ADR-0002、0009——此票不通过不得进入后续实施）。

**Blocked by:** 02 — 工具调用协议。

**Status:** ready-for-agent

- [ ] 冒烟脚本可手动触发，输出握手保真结论（SignalR 与 DOM 两通道分别验证）
- [ ] 实测并记录：单条消息字符上限、会话字符预算基线、单会话轮次上限
- [ ] 测得数据回填 config 默认值（maxMessageChars、sessionCharBudget 等）
- [ ] 协议格式（代码围栏包裹方案）用真机数据确认或修订，结论写入 ADR-0002 状态

# 02 — 工具调用协议（编解码与纠错）

**What to build:** 给定任意模型回复文本，协议层能可靠判定它是工具调用、完成声明还是 malformed，并生成协议提示词与纠错消息；整个协议默认包裹在代码围栏内（ADR-0002）。

**Blocked by:** 01 — Provider 抽象与 runtime 迁移。

**Status:** done

- [x] 协议提示词生成：工具清单 schema、输出格式规则（代码围栏包裹 + `<<<TOOL>>>`/`<<<DONE>>>`）、工作区信息、迭代规则
- [x] 解析器处理双层结构；外层代码围栏未闭合 → 判定回复不完整（触发续写而非解析）
